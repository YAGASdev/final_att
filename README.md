# Work Tracker - Учёт рабочего времени и задач

Настольное приложение для учёта сотрудников, проектов и задач с функциями аналитики.

## Описание

Приложение позволяет:
- Управлять списком сотрудников (добавление, редактирование, удаление)
- Вести учёт отработанных часов
- Рассчитывать заработную плату
- Создавать проекты и задачи
- Назначать задачи сотрудникам
- Отслеживать прогресс выполнения проектов
- Анализировать данные из CSV файлов
- Извлекать email-адреса из текста

## Установка

1. Клонируйте репозиторий
2. Создайте виртуальное окружение:
python -m venv venv

- source venv/bin/activate  # Linux/Mac
- venv\Scripts\activate     # Windows
3. Установите зависимости:
pip install -r requirements.txt

## Запуск
python main.py

## Технологии:
- Python 3.8+
- Tkinter (GUI)
- SQLite (база данных)
- Pandas (анализ данных)
- Регулярные выражения (обработка текста)
